# Practicum_diploma_qa

## О проекте
В рамках проекта проводилось тестирование веб-приложение по аренде самокатов Яндекс.Самокат, мобильного приложения для курьеров и тестирование API.

## Цель тестирования
Проверка функциональности, корректности работы форм, API и бизнес-логики.

## Типы тестирования
- Функциональное
- Негативное
- Регрессионное
- Smoke
- Кроссбраузерное
- UI
- Проверка валидации форм
- Тестирование мобильного приложения
- Тестирование API

## Используемые инструменты
- DevTools
- Postman
- Cygwin64 Terminal
- Android Studio
- Figma
- Google Таблицы

## Результат
- Составлены тест-кейсы
- Подготовлены чек-листы
- Найдены и оформлены баг-репорты
